GitHub via de Terminal tutorial

De handleiding GitHub via de Terminal is geschreven door één van onze Belgische leden, Stefan Adams. GitHub via de Terminal is misschien wat lastiger dan alleen pushen en pullen en het aanmaken van branches in Xcode. Volgens onze experts werkt GitHub beter én met meer mogelijkheden als je er een beetje ervaring in krijgt. Mocht je vragen hebben over deze handleiding, laat het dan via het chatkanaal aan Stefan (Adjiiij) weten.
